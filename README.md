## 文件过滤&行数统计
将某一路径下的所有指定类型的文件备份，保留原始的目录结构。可进行行数统计。
首次弹窗选择需要进行操作的文件夹，第二次弹窗选择备份的目标路径。

### 功能控制开关
````
COUNT_LINES = True  # 是否统计行数
COPY_FILE = True  # 是否备份文件
extlist = [".cpp", ".c", ".v", ".h", ".md", ".pdf"] # 白名单列表
countlist = [".cpp", ".c", ".v", ".h"] # 需要统计行数的文件类型
````
*行数统计不适用于python java等*
